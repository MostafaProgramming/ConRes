How To Read This

Marker-friendly interpretation

The Session Gate proves semaphore-based admission control. The Waiting Queue shows blocked users. The Readers lane highlights safe overlap, while the Writer Lock makes exclusivity obvious.

The timeline and event feed give a clear audit trail for fairness, bounded concurrency, and coordinated access to the shared file.

How To Use Manual Mode

Interactive concurrency walkthrough

Use Attempt Login to test the semaphore limit. Once the active slots are full, extra users will queue until someone logs out.

Use Read Start to show overlapping readers, then queue a writer to demonstrate how exclusive access blocks new reads until the writer lock is released.
